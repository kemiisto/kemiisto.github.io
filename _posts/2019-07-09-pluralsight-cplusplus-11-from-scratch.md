 - **Title**: [C++11 from Scratch][course]
 - **Author**: Giovanni Dicanio
 - **Publisher**: Pluralsight
 - **Year**: 2017
 - **Duration**: 5h 48m

 - **Rating**: ★★★★★
 - **Recommended**: as an introduction to modern C++ for beginners in programming.
 - **Not recommended**: for experienced programmers.

However strong is my objection to the idea of using C++ as the *first* programming language is, I must admit that the author of this course did a great job in introducing *some* important *practical* programming concepts for a beginner in the field using the modern C++. In particular, a viewer is introduced to the fundamental concepts of programming, such as, types, variables, functions, program flow control, user-defined types. Besides, a viewer is familiarized with some concepts of programming specific to C++, such as, preprocessor, header files, C++ parameter passing, std::string and std::vector standard classes.

Appreciating the efforts made by the author in introducing programming using C++ I have a very high opinion on this course, yet I still recommend to start learning programming from scratch using a different language, way simpler than C++.
